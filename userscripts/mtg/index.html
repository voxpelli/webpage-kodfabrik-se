MagicCards.info Cardlinker

  1. Install the Firefox extension Greasemonkey and restart Firefox
  2. Install the MTG Cardlinker Userscript [MagicCards.info-edition] (Help)
  3. Perhaps change which sites the MTG Cardlinker is activated on (Help)
  4. Test the script on something, new links have a yellow background.

Notes

  • Links cards to MagicCards.info cardpopup and by default checks for card names on StarCityGames.com and MTG Salvation, please suggest more sites.
  • This userscript checks for a new version on this server once a week, max, to notify you about updates.
  • You're using this userscript at your own risk - of course.

MTG Cardlinker

  1. Install the Firefox extension Greasemonkey and restart Firefox
  2. Install the MTG Cardlinker Userscript (Help)
  3. Perhaps change which sites the MTG Cardlinker is activated on (Help)

SvM Cardlinker

  1. Install the Firefox extension Greasemonkey and restart Firefox
  2. Install the MTG Cardlinker Userscript [SvM-edition] (Help)
  3. Perhaps change which sites the MTG Cardlinker is activated on (Help)
  4. Test the script on something, for example this old Nemesis-article, new links can be identified by their yellow background.

Notes

  • Links cards to SvenskaMagic.com's cardpopup and by default checks for card names on that site only.
  • This userscript checks for a new version on this server once a week, max, to notify you about updates.
  • You're using this userscript at your own risk - of course.
