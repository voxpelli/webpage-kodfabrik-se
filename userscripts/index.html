What is this?

These are some scripts that I've coded for one reason or another and decided to release for free. Some I've coded because they were fun to code, some because I wanted the functionality myself and some because I thought that I should give something back to the open source community.

Greasemonkey

Userscripts for the Firefox extension Greasmonkey.

  • MTG Cardlinker - a userscript that searches a site for the names of cards from the Magic: The Gathering trading card game and links these to sites where information about the card can be found.

Operator

Scripts that adds additional functionality to the Firefox extension Operator, which attaches actions to metadata embedded into sites with microformats and similar.

To install a script you have to download it to your computer first and then open up the settings dialog for the Operator extension. Choose the tab "user-scripts" and click on the button "new". Select the file you just downloaded and it will be installed.

After that you move over to the "actions" tab and once again you click on "new" and selects one of the new actions that has been added. Then you're done :)

  • Swedish Operator Kit 0.8 - adds several actions that are useful to swedish users, for example tags on Bloggar.se and maps on Eniro.se.

DOMAssistant

I'm a developer of DOMAssistant itself but since DOMAssistant should stay slim far from all functionality are fit to go into the core, therefor I've coded these plugins.

  • DOMEffects (alpha/beta) - an effects plugin for DOMAssistant enabling the animation of elements as well as some shortcuts to usual effects.
  • DOMValidator (beta) - a validator plugin that enables you to add rules to forms and having it automatically adding classes and error messages when validated - which can be done on form submit for example. Works nicely with Robert Nymans DOMAssistantAJAXForms
